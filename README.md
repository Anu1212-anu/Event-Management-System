# Event Management System
Full Stack web application to manage events, bookings and users.

## Tech Stack
Java | Spring Boot | REST APIs | MySQL | React

## Features
- User registration & login
- Create and manage events
- Ticket booking
- Admin dashboard
- CRUD operations

## My Role
- Developed backend APIs using Spring Boot
- Designed MySQL database
- Integrated frontend with REST APIs
- Tested complete application

## How to Run
1. Start backend server
2. Start frontend
3. Configure MySQL
4. Run project
